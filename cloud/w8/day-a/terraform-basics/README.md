# Terraform Basics Sandbox

Bộ file này dùng để luyện cú pháp HCL cho W8-D1 mà không cần tạo tài nguyên AWS thật.

## Nội dung

- `versions.tf`: version constraint
- `variables.tf`: biến đầu vào và validation
- `locals.tf`: map/list/object và `for` expression
- `outputs.tf`: đầu ra để quan sát cách Terraform evaluate expression
- `terraform.tfvars.example`: ví dụ override biến
