# AWS Learning Platforms

## Purpose

This note helps you choose between AWS learning paths and AWS hands-on labs. The key question is whether you need structured theory first or task-driven practice first.

## What To Remember

- `AWS Skill Builder` is a better fit for structured learning paths and foundational study.
- `AWS Workshops` are better for guided hands-on exercises with clear outcomes.
- Workshops can generate AWS costs because they may create real resources in your account.
- It is usually not worth jumping into AWS workshops if your local Terraform and Kubernetes foundation is still weak.

## When To Use Which Platform

- if you need a structured path
  - use `Skill Builder`
- if you need a guided hands-on scenario
  - use `Workshops`

## How It Applies to W8

- W8 should prioritize local foundations first.
- After W8, workshops become more useful for areas such as:
  - IAM
  - networking
  - containers
  - observability
  - platform engineering

## Important Warnings

- a free workshop page does not always mean the hands-on portion is free to run
- always read cleanup instructions
- if you use a personal account, control region choice and cost carefully
