## Minimum Theory You Must Know

- GitOps principles: declarative desired state, versioned state, reconciliation, drift correction.
- Argo CD: Application, sync, health, app-of-apps, sync waves.
- CI/CD: plan or validate on PR, apply or sync after merge.
- Observability: metrics, logs, traces, dashboards, alerts.
- OpenTelemetry: SDK instrumentation and Collector pipelines.
- Prometheus: scraping, labels, PromQL, alert rules.
- Grafana: dashboard, datasource, panel, query.
- Loki: log aggregation and label-based querying.
- SLO/SLI: availability, latency, error budget.
- Burn rate: fast and slow multi-window alerts.
- Canary: progressive traffic exposure and automatic abort.
- Argo Rollouts: Rollout CRD, steps, pause, analysis.
- k6: VUs, duration, checks, thresholds.
